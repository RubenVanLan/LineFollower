# Sensoren proof of concept

minimale hard- en software die aantoont dat minimaal 6 sensoren onafhankelijk van elkaar kunnen uitgelezen worden (geen calibratie, normalisatie of interpolatie). Hierbij moet een zo groot mogelijk bereik van de AD converter benut worden (indien van toepassing)

Sluit aan volgens het elektrische schema en laad de code op.
Je om de waarden van de sensor te bij een zwarte (de lijn) of een witte ondergrond te zien, kan je een wit/zwart blad onder de sesor schuiven. Hou de sensor ope een stabiele op hoogte (1 Ã  2 cm).
